# my-book-sales-page_-
„Изграден отвътре“ е практическо ръководство за мъже между 40 и 60 години, които търсят нова посока, повече яснота и смисъл в живота си. Книгата включва методология за самоанализ, практически упражнения, 30-дневен план за действие и стратегии за личностно развитие и уверен напредък.
